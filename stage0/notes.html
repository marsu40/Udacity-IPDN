The web & HTML basics

The world wide web It was developed in the early 1990's. It is a collection of html documents. html stands for HyperText Markup Language which is the basis for almost any webpage. You can find several types of documents on the web (images, videos, music,...). HTML allows to organize all those things together. The hyperlinks allow to link the pages to one another.
Their are 3 major components of the web :
  • A PC running a web browser (ex. Firefox, Chrome)
  • The internet
  • Servers (computers like a normal PC but dedicated for hosting files)
the PC running the browser, to be able to communicate with the server uses a simple protocol called HTTP.
HTML
An HTML page is composed of:
  • Text content
  • Markup (tags allowing to modify the way the text or other elemnts appears)
  • References ot other documents (images, videos, sounds, ..)
  • Links to other pages (hyperlinks)
HTML Elements

HTML Elements correspond to the markup par of HTML.
They allow to tell the web browser how to display the page.
They most of the time consist in an opening tag and a closing tag (ex. <b> content <b>).
When there is no closing tags, the tag is called void tag (ex. <img>).

HTML Attributes

Some elements can include values which are called attributes.
They provide additional information to the browser.
For example: <img src="url" alt="text"> Displays the image hosted at the src attribute value "url".
If the image cannot be displayed, then the alt attribute value "text" is shown. We have also to differenciate "inline" from "block" elements.
Inline elements do not have any specific behavior on the content they encompas between their opening and closing tags.
On the other hand, block elements, create a virtual "box" around the content between their opening and closing tags.
